# 🔮 MysticHub — сервіс персональних передбачень

## 📖 Про проєкт

MysticHub — це веб-сервіс, який надає користувачам можливість отримувати персональні астрологічні передбачення.  
Платформа поєднує елементи гороскопів, нумерології та випадкових передбачень у зручному та інтерактивному форматі.

=======
Основна ідея сервісу полягає у створенні простого та привабливого веб-інтерфейсу, де користувач може дізнатися передбачення на день або отримати гороскоп для свого знаку зодіаку.  
У майбутньому функціонал може бути розширений аналізом сумісності, нумерологічними розрахунками та додатковими астрологічними інструментами.

Система працюватиме на основі введених користувачем даних або вибору відповідних параметрів (наприклад, знаку зодіаку).  
На основі цього сервіс генеруватиме передбачення, використовуючи прості алгоритми визначення знаку зодіаку та випадкову генерацію прогнозів.

Проєкт включає:

- **Frontend-частину** — інтерфейс сайту для взаємодії з користувачем
- **Backend-логіку** — обробку даних та генерацію передбачень

## 🚀 MVP (Мінімально життєздатний продукт)

Мінімальний функціонал сервісу включає:

### 1. Головна сторінка
Відображення головного екрану сайту з інтерактивною картою передбачення.

### 2. Передбачення на день
Можливість натиснути на карту та отримати випадково згенероване передбачення на поточний день.

### 3. Навігація сайту
Реалізація навігаційного меню для переходу між сторінками:

- Головна  
- Гороскоп  
- Сумісність  

### 4. Сторінка гороскопу
Відображення всіх **12 знаків зодіаку** у вигляді інтерактивних карток.

### 5. Вибір знаку зодіаку
Можливість обрати знак зодіаку та перейти до перегляду відповідного прогнозу.

### 6. Відображення прогнозу
Виведення текстового гороскопу для обраного знаку.

### 7. Інформаційний футер
Футер сайту з інформацією про:

- соціальні мережі  
- контактні дані  
- команду розробників  


## ⚡ Майбутній функціонал

У подальшому планується розширення функціоналу сервісу:

### 1. Аналіз сумісності
Можливість ввести імена та отримати оцінку їх сумісності.

### 2. Нумерологічні розрахунки
Обчислення особистих числових показників на основі дати народження.

### 3. Розклад таро
Генерація віртуального розкладу карт таро для отримання додаткових передбачень.

### 4. Персоналізація
Отримання більш індивідуальних прогнозів на основі введених даних користувача.
